Client Worker Required

The client worker service must be running before you can access the dashboard.

To get started:
  1. Open a terminal/command prompt
  2. Navigate to the client_worker directory
  3. Run: python client_service.py --server http://192.168.88.101:8000
  4. Keep the client worker running
  5. Refresh this page once the worker is connected
